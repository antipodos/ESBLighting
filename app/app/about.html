• What?

    ESB Lighting provides a schedule of the upcoming (and past) Empire State Building lightings.
    Per default all available future data as well as past data (up to the last month's first day) is shown. You can load all available data on request (ESB provides a one year's history).

  • How?

    The information is pulled from the official ESB website and provided as a seperate service. This app consumes that service and displays the results.
